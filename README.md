# 🛠️ JSONTracer Pro
**Professional JSON Formatter & Debugger for Transaction Logs**

JSONTracer Pro adalah alat web sederhana namun kuat yang dirancang khusus untuk membantu analis (seperti System Test Analyst) dalam melakukan tracing log transaksi, debugging API, dan analisis data JSON secara cepat dan aman.

## ✨ Fitur Unggulan
- **Two View Modes**: Pilih antara `Beautify` (format rapi) atau `Compact` (format ramping vertikal).
- **Syntax Highlighting**: Pewarnaan otomatis untuk Key, String, Number, dan Boolean untuk memudahkan pembacaan log.
- **Dark & Light Mode**: Navigasi nyaman di berbagai kondisi pencahayaan.
- **Secure by Design**: Semua proses dilakukan di sisi klien (Client-side). Data Anda tidak pernah dikirim ke server mana pun.
- **Fast Export**: Simpan hasil format langsung ke file `.txt` dengan satu klik.

## 🚀 Cara Penggunaan
1. Buka https://frhndwptra.github.io/json-formatter-pro/
2. Paste log JSON yang berantakan ke kolom **Source JSON**.
3. Pilih mode yang diinginkan.
4. Salin hasil atau simpan sebagai file teks.
